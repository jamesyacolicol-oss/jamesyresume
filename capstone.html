Project Rationale

💡
The Problem

Manual classification is prone to human error. This system automates the process using Deep Learning, reducing misidentification and improving agricultural efficiency.

🎯
Objective

To provide local farmers and agricultural students with an accessible tool for identifying coconut seedling varieties to ensure better crop yield.

Development Stack

Mobile Frontend Flutter / Dart
Web & API Backend Laravel 10 / PHP
AI Engine Python / Flask
Model Type TensorFlow (CNN)
Database MySQL (Clever Cloud)
